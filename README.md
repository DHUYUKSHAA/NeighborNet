# 🛡️ NeighborNet - AI-Powered Network Security Monitoring System

NeighborNet is an AI-powered network security monitoring application that analyzes network traffic, detects suspicious activities, and provides real-time insights through an interactive dashboard. The system helps users monitor network behavior, identify potential threats, and improve overall network security.

---

## 📌 Features

- 🔍 Real-time network traffic monitoring
- 🤖 AI-based suspicious activity detection
- 📊 Interactive dashboard for network analysis
- 📈 Traffic visualization and statistics
- 🚨 Threat detection and alerts
- 📁 Network log management
- 📋 Detailed security reports
- 💻 User-friendly interface

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/DHUYUKSHAA/NeighborNet.git
```

### 2. Navigate to the project

```bash
cd NeighborNet
```

### 3. Create a virtual environment (Optional)

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux / Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the application

```bash
python app.py
```

### 6. Open in browser

```
http://127.0.0.1:5000
```

## ⚙️ How It Works

1. User starts the application.
2. Network traffic is collected.
3. AI model analyzes the traffic.
4. Suspicious activities are detected.
5. Dashboard displays analysis and alerts.
6. Security reports are generated for review.
